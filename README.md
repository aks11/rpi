# rpi
Python scripts for interfacing RaspberryPi to pheripherals/devices like
1.] Buzzer - CHB04
2.] LCD - JHD204a
3.] Memory - AT24C512
4.] RTC - PCF8563

I[maybe YOU] will continue to add new device scripts.

### Note
THESE SCRIPT'S WERE TESTED ON RASPBERRY-PI3 ONLY.
